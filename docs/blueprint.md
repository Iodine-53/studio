# **App Name**: ToolboxAI

## Core Features:

- Homepage Dashboard: Interactive dashboard presenting a range of tools: document creation, spreadsheet generation, file format conversions, and PDF manipulation.
- Document Maker: Directly create and export text documents.
- Spreadsheet Generator: Directly generate and export spreadsheets with basic functionality.
- Image Converter: Convert images between formats like JPEG, PNG, and WebP.
- Audio Converter: Convert audio files between formats like MP3, WAV, and AAC.
- Video Converter: Convert video files between formats like MP4, MOV, and AVI.
- PDF Manipulator: Perform PDF manipulations: merge PDFs, add watermarks, compress files.
- AI Brainstorming: AI for brain storming

## Style Guidelines:

- Primary color: A vibrant sky blue (#4682B4), reminiscent of open skies and limitless possibilities, embodying the range of tools available.
- Background color: A light, desaturated blue-gray (#E6E9ED), offering a clean and modern backdrop that emphasizes the vibrant interface elements.
- Accent color: A contrasting soft yellow (#EBD979), used for call-to-action buttons and interactive elements, providing a clear visual distinction without overwhelming the user.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, paired with 'Inter' (sans-serif) for body text, which offers a balance of modern aesthetics and readability for all types of content.
- Use minimalist, line-style icons that are intuitive and universally recognizable to represent each tool and function. Ensure consistency in style and weight.
- A clean, grid-based layout ensuring balanced spacing and clear visual hierarchy. Use of white space to separate sections and tools.
- Subtle transitions and animations when navigating between tools or initiating a conversion process, providing feedback without distracting the user.
